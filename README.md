# Customizing Your Website

## Instructions for Customization
1. **Clone the Repository**: 
   - Use the command `git clone https://github.com/sarahlouise0589/SLRushworth.git`

2. **Edit the HTML/CSS**: 
   - Open the cloned folder and navigate to the `index.html` file.
   - Modify the HTML structure and styles in `style.css` as needed.

3. **Add Your Content**:  
   - Replace placeholder text with your own in the HTML files. 
   - Add any images or other media in the `assets` folder.

## Setting Up GitHub Pages Hosting
1. **Navigate to the Repository on GitHub**: 
   - Go to `https://github.com/sarahlouise0589/SLRushworth`

2. **Go to Settings**:  
   - Click on the `Settings` tab in your repository.

3. **Scroll to GitHub Pages**: 
   - Under the `Pages` section, choose the source branch (usually `main`) and select `/ (root)` as the folder.

4. **Save the Changes**:  
   - Click `Save`, and GitHub will publish your site. 

5. **Visit Your Website**:  
   - Your website will be available at `https://sarahlouise0589.github.io/SLRushworth/`.

## Customizing the Template for Personal Use
1. **Choose a Color Scheme**:  
   - Update colors in the `style.css` file to match your personal branding.  

2. **Change Fonts**:   
   - Link to your preferred fonts in the HTML head or include them in the `style.css` file.

3. **Adjust Layouts**:  
   - Modify sections in the HTML to rearrange items as per your needs.

4. **Test Your Changes**:  
   - Open the HTML files in a browser to review adjustments before pushing to GitHub.

5. **Push Changes to GitHub**:  
   - Use `git add .` followed by `git commit -m "Update website"` and `git push` to upload your changes.
